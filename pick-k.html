Pick K

This was a problem I saw posted on the /r/learnprogramming subreddit. A self-taught dev had been given this as part of their first technical interview. It seemed doable at my level (though other things they were asked during the interview are beyond me right now) so I thought I'd take a swing at it.

The problem

Given an array of distinct n numbers, randomly pick k distinct numbers from the list. You are only given k chances to pick the number. You are not allowed to have temporary storage (like another list).

e.g. [1,6,5,3] n=4 k=2
result [1,6] or [1,5] or [5,6] etc

Let's decompose the problem...

  • Choose a random position p of an array
    • Choose a random number between 1 and n
  • Retrieve the number at position p
  • Amend an array to remove a number at a specified position
  • Add the removed number to a new array k
  • Loop all of this k times

My solution

It's running on this page, open dev tools and check the console. You can also call the function by typing pickK([x], k) into the console, where x is the array.
